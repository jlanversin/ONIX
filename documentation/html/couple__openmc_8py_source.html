22 class Couple_openmc(object):
23  """This class is used to execute coupled-mode simulations
24 
25  Through this class, the user can:
26  - chose to parallelize the Monte Carlo simulations
27  via the "set_MPI" method,
28  - set the nuclear data libraries to be used in the simulation,
29  - set the burnup/time sequence for the simulation,
30  - manually set the volume of each BUCells (note that by default, ONIX uses the OpenMC stochastic volume calculation
31  to find the volume of each cells. However, this method might produce important errors on the volume and it is
32  advised to set the volume manually)
33  - select which of the Cells defined in the OpenMC input should be depleted (known as BUCells)
34  - select a list of nuclides in each BUCell which cross sections will be tallied. If the user
35  does not specify any such list of nuclides, ONIX will by default tally the cross sections of all
36  nuclides which data are found in the cross section library.
37 
38  The Couple_openmc class also takes care of the coupling between ONIX and OpenMC. At the beginning of a simulation,
39  it imports certain initial parameters from the OpenMC input to ONIX such as the name of the BUCells and the initial
40  nuclide densities of each BUCell. During the simulation, it makes sure that information such as the flux tallies,
41  neutron spectrum tallies, the reaction rates and the updated nuclide densities are correctly transfered between the burnup
42  module and the neutron transport module.
43 
44  The Couple_openmc class will also sample the isomeric branching ratios and (n,gamma) cross sections on the
45  the same energy points to prepare for the calculations of one-group isomeric branching ratios.
46 
47  IMPORTANT: the root cell in the OpenMC input should be named "root cell" """
